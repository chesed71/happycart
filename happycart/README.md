# happycart (해피카트)

한국어 식품 바코드 스캐너 MVP. Flutter + Supabase 백엔드.

## 개발 환경

- Flutter 3.41.9 (stable)
- Dart 3.11.5

## Flavor

`development`, `staging`, `production` 3개의 빌드 flavor를 지원해요.

```
flutter run --flavor development
flutter run --flavor staging
flutter run --flavor production
```
